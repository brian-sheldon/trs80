# Z80 reference and test files

## zexdoc.com / zexall.com

Z80 instruction set exerciser programs for CP/M, used to validate Z80
emulator correctness. These are **not** original work of this repository.

- **Original author:** Frank D. Cringle, as part of
  [YAZE (Yet Another Z80 Emulator)](https://github.com/begoon/z80exer).
- **CP/M port:** J.G. Harston (2002), modified to assemble with ZMAC
  and MAXAM. Available at <https://mdfs.net/Software/Z80/Exerciser/>.
- **License:** GNU General Public License (GPL).

ZEXDOC tests documented flag behavior (67 instruction groups). ZEXALL
additionally tests the undocumented X and Y flags (bits 3 and 5).

## z80_reference.md

A Z80 instruction set reference compiled for emulator development.
Factual hardware documentation created for this project.
